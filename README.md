# Health IoT Monitor 🫀

## The Problem
Real-time health monitoring doesn't exist for the general public.
Hospitals monitor patients — nobody monitors humans in their daily life.

## The Solution
A connected subcutaneous sensor that streams biological constants in real-time :
- Heart rate & HRV
- Blood oxygen (SpO2)
- Body temperature
- Blood pressure
- Sugar rate
Connected to an app with intelligent alerting when something is wrong.

## Why I'm building this
I worked in an operating room. I prepared respirators, monitored patients,
assisted with intubations. I know what these constants mean in real life.
This project is built from field experience, not theory.

## Stack
- Backend : Python / Django / PostgreSQL
- IoT : MQTT protocol / sensor simulation
- Mobile : Flutter
- AI Alerting : coming soon
